# Setup
1. Make a virtualenv
```sh
virtualenv flaskenv
```
2. start the virtualenv
linux-
```sh
source flaskenv/bin/activate
```
windows-
```sh
flaskenv/Scripts/activate
```
3. Install dependencies
```sh
pip install -r requirements.txt
```
4. Run the server
```sh
python main.py
```
